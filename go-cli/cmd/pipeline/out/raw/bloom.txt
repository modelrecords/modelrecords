Model Details:
  Name: BLOOM
  Version: 1.0.0
  Release Date: Monday, 11.July.2022
  Developed By: BigScience
  Model Type: Transformer-based Language Model

Intended Use:
  - Text generation
  - Exploring characteristics of language generated by a language model
  - Information Extraction
  - Question Answering
  - Summarization

Factors:
  - Language
  - Domain
  - Demographic characteristics

Metrics:
  - Perplexity
  - Cross Entropy Loss

Evaluation Data:
  Description: The model was evaluated using the humaneval dataset, designed to test a variety of capabilities in Python programming tasks.

Training Data:
  Description: Training data includes 46 natural languages and 13 programming languages, comprising 1.6TB of pre-processed text converted into 350B unique tokens. 

Ethical Considerations:
  - Overrepresentation of some viewpoints and underrepresentation of others
  - Potential generation of hateful, abusive, or violent language

Caveats and Recommendations:
  - Users should be aware of Risks and Limitations, including an appropriate age disclaimer as necessary.
  - Models trained or finetuned downstream of BLOOM LM should include an updated Model Card.
  - Users of the model should provide mechanisms for feedback.

Additional Information:
  - The model is not designed for critical decisions or uses with material consequences on an individual's livelihood or wellbeing. 
  - Estimated carbon emissions and electricity usage for training the model are forthcoming.
